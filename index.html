DATA SOURCES & INFORMATION

Programs & Attributions

This map was made by Dilni Abeyrathne for the New Maps Plus program (MAP671) using QGIS v.3.22.13. The haunted location data was compiled using Microsoft Excel and coordinates were retrieved from Google Maps. Additionally, this map drew inspiration from a previous map made by Dilni Abeyrathne exploring popular haunted and abandoned locations in Kentucky. The coordinate projections used for this project are: ESPG:4326 (for the haunted sites only) and ESPG:3089 (for all other data).

Data Sources

The majority of haunted sites was compiled using information from Kentucky Haunted Houses. Supplementary data for the Kentucky State Penitentiary, Camp Taylor, Sleepy Hollow Road, and Asbury Cemetery were sourced from these sites: KY State Penitentiary & Camp Taylor, Sleepy Hollow Road, and Asbury Cemetery. Additional references and information were sourced from the following reddit threads: Haunted Places and Haunted Areas. Finally, additional sites in Lexington and Kentucky were sourced from Haunted Rooms (Lexington, Kentucky) and Haunted Rooms (Kentucky).

Finally, Kentucky state and county polygon data was obtained from the U.S. Census Bureau, Kentucky roads and rivers from Natural Earth Data, and the ghost raster image marker from Dilni Abeyrathne's previous spooky mapping project.
